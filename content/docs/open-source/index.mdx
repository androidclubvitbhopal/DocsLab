# Getting Started with Open Source

Welcome to the open-source initiative of the Android Club! This section is designed to help you make your first contribution, understand standard open-source workflows, and get comfortable with tools like Git and GitHub.

## Why Contribute?

- **Learn by doing:** Work on real-world projects instead of just tutorials.
- **Build your resume:** Public contributions are visible to recruiters and showcase your skills.
- **Community:** Join a network of passionate developers and mentors.
