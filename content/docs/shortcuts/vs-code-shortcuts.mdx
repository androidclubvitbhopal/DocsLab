# VS Code Shortcuts

VS Code is our recommended editor for web development. Here are essential shortcuts:

## General

- **Command Palette:** `Ctrl` + `Shift` + `P` (Windows/Linux) / `Cmd` + `Shift` + `P` (Mac)
- **Quick Open (Go to File):** `Ctrl` + `P` (Windows/Linux) / `Cmd` + `P` (Mac)
- **Toggle Terminal:** `Ctrl` + `` ` `` (Backtick)

## Editing

- **Duplicate Line Down:** `Shift` + `Alt` + `DownArrow` (Windows/Linux) / `Shift` + `Option` + `DownArrow` (Mac)
- **Move Line Up/Down:** `Alt` + `UpArrow`/`DownArrow` (Windows/Linux) / `Option` + `UpArrow`/`DownArrow` (Mac)
- **Multi-Cursor:** `Alt` + `Click` (Windows/Linux) / `Option` + `Click` (Mac)

## File Management

- **Toggle Sidebar:** `Ctrl` + `B` (Windows/Linux) / `Cmd` + `B` (Mac)
- **Find in Files:** `Ctrl` + `Shift` + `F` (Windows/Linux) / `Cmd` + `Shift` + `F` (Mac)
