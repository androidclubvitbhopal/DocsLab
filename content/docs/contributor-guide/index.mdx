# Contributor Guide

Thank you for your interest in contributing to Android Club VITB!

## Code of Conduct

Please be respectful to all members and contributors. We value a collaborative and inclusive environment.

## Communication

- We use our **Discord Server** for all project-related discussions.
- Create an Issue on GitHub before opening a Pull Request (PR) to discuss the changes you want to make.

## Writing Documentation

Our documentation uses MDX. You can include React components directly inside your Markdown files!

```mdx
import { Callout } from "fumadocs-ui/components/callout";

<Callout title="Note">This is a callout block!</Callout>
```
