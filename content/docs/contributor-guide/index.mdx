# Contributor Hub

Thank you for your interest in contributing to Android Club VITB! This hub contains everything you need to know about contributing content, code, or documentation to our projects.

## Code of Conduct

Please be respectful to all members and contributors. We value a collaborative and inclusive environment.

## Communication

- We use our **Discord Server** for all project-related discussions.
- Create an Issue on GitHub before opening a Pull Request (PR) to discuss the changes you want to make.
